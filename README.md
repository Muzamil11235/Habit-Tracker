# Habit Tracker

A simple and user-friendly **Habit Tracker** application built in **Java**. This project helps users create, manage, and track their daily habits to build consistency and improve productivity.

## Features

- Add new habits
- View all habits
- Update habit details
- Delete habits
- Track daily progress
- Simple and easy-to-use interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling (if used)
- Java Swing / JavaFX (if your project has a GUI)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Muzamil11235/Habit-Tracker.git
```

2. Open the project in your preferred Java IDE (VS Code, IntelliJ IDEA, Eclipse, or NetBeans).

3. Build and run the project.

## Usage

- Launch the application.
- Create a new habit.
- Track your daily progress.
- Update or remove habits as needed.
